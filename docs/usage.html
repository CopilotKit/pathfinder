Getting Started

Two install paths. Same server. Choose what fits your workflow.

Quick Start

CLI

$ npx @copilotkit/pathfinder init $ npx @copilotkit/pathfinder serve

Quickest start. No Docker needed. Add Postgres later for semantic search.

Docker

$ docker pull ghcr.io/copilotkit/pathfinder $ docker run -v ./pathfinder.yaml:/app/pathfinder.yaml \ -v ./docs:/app/docs -p 3001:3001 \ ghcr.io/copilotkit/pathfinder

Production-ready. Use docker-compose to add Postgres for full semantic search.
